Simple present



Welcome congratulations because you've given the firs step.

Today we'll learn about the simple present and we also learn new vocabulary and verbs so let's start. First of all What is the Present simple? the present simple is the most basic tense but why?.


1. we use the simple present when something is generally or always true.

Example:

-we drink water (we know that we need drink water.)

2. we use for habits or things that we do regulary or always.

Example:

-I do exercise everyday.

3. We can also use for things that are happening now.

Example:

-Tom plays with his friend.

Now that we know when we use the simple present what is the structure?


Structure


With third person



Doesn't change but with the third person(She/He/It) singular we only add an -s or -es the end of the verb.

Example:

-Mary(She) lives in london.

-Tom(He) works in a big company.

-the lion(It) eats meet.

if the verb ends in: o, s, sh, ch, x or z should be added to it -es.

when the verb ends in: y should be added to it -ies

the rest are with -s but there are exceptions like have=has in third person.
